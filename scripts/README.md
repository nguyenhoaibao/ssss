# Migration script

### How to run

```bash
$ node migrate_from_old_WP.js
```

You also need to specify these arguments:

```
--wp-host
--wp-port
--wp-username
--wp-password
--wp-dbname
--api-host
--api-port
--api-username
--api-password
--api-dbname
```

Run script with `--help` to get more information.
